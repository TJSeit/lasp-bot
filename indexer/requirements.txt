# Indexer dependencies — run locally on a machine with an NVIDIA GPU
# Install with: pip install -r requirements.txt
#
# NOTE: The extra index URL below is required to install the CUDA-enabled
# PyTorch wheel.  Without it pip resolves torch from PyPI, which ships only
# the CPU-only build.  That causes torch.cuda.is_available() to return False
# even on a machine with a supported NVIDIA GPU (e.g. RTX 4070 Ti), and
# build_index.py fails when trying to initialise GPU embeddings.
